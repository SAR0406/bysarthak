# **App Name**: Sarthak's Spectrum

## Core Features:

- Animated Hero Intro: Full-screen intro with kinetic typography revealing the name 'Sarthak'. Features organic morphing shapes.
- Dynamic Tagline: Display a dynamic tagline, such as 'Creative Coder & Explorer of Modern Web Experiences'.
- Smooth Scrolling: Implement 'Explore My World' button that triggers smooth scroll animations to different sections.
- Sticky Navbar: Transparent navbar that turns solid with smooth color changes on scroll.
- GitHub Integration: Embed GitHub profile (https://github.com/SAR0406) with animated loading of repositories. Displays top repos as animated cards.
- Parallax Contact Form: A parallax scrolling contact form section with floating social media icons with animated transitions.
- Theme Toggling: Integrate a dark/light theme toggle with smooth color morphing. Uses clean, modern typography.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke a sense of futuristic elegance.
- Background color: Very light grey (#F0F0F0), almost white, to ensure readability and a clean aesthetic. Suitable for a light scheme.
- Accent color: Electric purple (#8F00FF), for hover states, highlights, and interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for a techy, scientific feel. Body font: 'Inter' (sans-serif) to support longer text.
- Use minimalistic icons with subtle hover effects to maintain a clean interface.
- Ensure full responsiveness using CSS Grid and Flexbox for different screen sizes.
- Employ subtle micro-animations on hover and smooth transitions to improve UX.
